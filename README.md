# AI-SOC-alert-analysis
AI-powered SOC alert triage system using Aria AI and Python

## 📌 Overview

This project demonstrates an AI-driven Security Operations Center (SOC) solution that automates alert triage using Aria AI. The system processes JSON-based security alerts and provides actionable insights to assist in threat detection and response.

## 🚀 Features

* AI-based alert analysis and classification
* Automated SOC alert triage
* JSON log processing
* API integration with Aria AI
* Real-time alert handling using Python

## 🛠️ Technologies Used

* Python
* Linux
* Aria AI
* REST API
* JSON

## ⚙️ How It Works

1. Security alerts are generated in JSON format
2. Python script sends alerts to Aria AI via API
3. AI agent analyzes the alert
4. Returns threat insights and recommendations

## 🔍 Sample Use Cases

* Brute force attack detection
* Suspicious login identification
* Anomaly detection in network activity

## 📊 Example Alert

```json
{
  "event_type": "failed_login",
  "source_ip": "192.168.1.10",
  "attempts": 15
}
```

## 📈 Output

AI identifies potential brute force attack and flags IP as suspicious.
